# truthonly-xhs-cover

truthonly 风格小红书封面提示词生成 Skill。

这个 Skill 用于把文章、选题、产品测评、AI 工具教程、效率方法、播客推荐、个人经验等内容，整理成可直接交给图片生成模型使用的小红书封面提示词。

核心视觉方向：真人出镜、超大柔和的浅黄（#FDFFA7）/白色中文标题、粗黑描边、人物抠图描边、绿色勾选清单、贴纸元素、箭头标注和高密度高对比构图。

## 适用场景
- AI 工具教程封面
- 产品测评和榜单封面
- 效率工作流封面
- 小白科普问答封面
- 播客、资源、工具种草封面
- 个人经验和方法论封面
- 从 0 搭建、部署、上手类教程封面

## 安装方式

把 `truthonly-xhs-cover` 目录复制到 Codex skills 目录：

cp -R truthonly-xhs-cover ~/.codex/skills/

$truthonly-xhs-cover

用 truthonly-xhs-cover 帮我生成一张 AI 工具测评小红书封面提示词。

使用流程

会按顺序向用户提问，并且一次只问一个问题：

选择封面风格
上传人物参考图
选择人物表情和动作
选择是否有额外素材图
选择背景色调
选择字体风格
选择字体颜色效果
从候选标题中选择最终标题
收集完信息后，Skill 会只套用用户选择的风格模板，输出一段完整、结构清晰、可直接用于图片生成模型的封面提示词。

封面风格

目前包含 8 种封面风格：

爆款大字压顶风
巨字拆分冲击风
小白科普问答风
教程清单风
产品测评榜单风
种草推荐风
贴纸拼贴入门风
黑底效率工作流风
设计规范

固定 3:4 竖版小红书封面比例
真人出镜，人物通常占画面 35%-55%
主标题必须超大、清晰、短促
主标题优先使用柔和浅黄色 #FDFFA7 或纯白，并带粗黑描边
标题填充、关键词、标题背后的底板/色块/横幅只要使用黄色，统一指定为 #FDFFA7
人物边缘需要明显抠图描边
画面强调高对比、高密度、强情绪和一眼可读
重要文字、人物脸部和清单内容不能超出安全区
