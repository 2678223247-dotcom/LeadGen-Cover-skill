# truthonly-xhs-cover

## 描述
truthonly 风格小红书封面提示词生成器。用于AI工具、效率方法、产品测评、教程部署、个人经验、播客推荐、职场认知类内容的爆款封面设计。

## 风格特征
- 真人出镜
- 超大柔和浅黄（#FDFFA7）/ 白色中文标题
- 粗黑描边
- 人物抠图描边
- 绿色勾选清单
- Emoji贴纸、箭头标注
- 高对比高密度构图


## 工作流程（严格按顺序执行）

**重要：一次只问一个问题，等用户回答后再问下一个。**

如果用户已在当前消息提供了某项信息，直接记录并跳过对应问题。

除用户粘贴文章、脚本、选题或上传图片外，所有封面决策问题都必须做成**编号选择题**。不要用开放式问题。


### 第1问：选风格

列出以下8种风格，让用户选一种：

1. **爆款大字压顶风** — 顶部超大标题，人物居中，适合强钩子标题
2. **巨字拆分冲击风** — 将标题拆解到不同位置，设计感强
3. **小白科普问答风** — 黑灰科技海报、模糊界面或真实室内背景，问号和解释感，适合"什么是/怎么理解"
4. **教程清单风** — 绿色勾选清单 + 步骤感，适合安装、部署、上手教程
5. **产品测评榜单风** — 大数字、箭头、维度标签，适合横评、榜单、测评
6. **种草推荐风** — 人物点赞或推荐姿势，emoji和温暖背景，适合资源/播客/工具推荐
7. **贴纸拼贴入门风** — 人物 + 小卡片 + 产品图标 + 贴纸，适合"入门/搭建/组合"
8. **黑底效率工作流风** — 深色背景 + Logo墙 + 清单，适合工作法、效率流、工具组合


### 第2问：上传人物参考图

要求用户上传人物参考图。

- 可以上传1张清晰人脸照
- 也可以上传2-3张同一人物照片用于保持人物一致性
- **如果用户明确说不露脸**：也必须要求上传局部出镜参考图（手部、半脸边缘或侧脸轮廓），不要凭空生成默认人物
- 用户上传后，只记录为图1人物参考，不额外追问人物外貌


### 第3问：选择人物表情和动作

让用户从以下选项中选择：

1. 微笑看镜头（亲和力强）
2. 手指指向侧方（引导视线）
3. 双手交叉胸前（专业感）
4. 侧身回眸（自然感）
5. 手持产品/道具展示
6. 不露脸（纯手势/局部特写）


### 第4问：选择是否有额外素材图

让用户选择：

1. 无额外素材，仅人物 + 标题
2. 有产品图（如工具截图、测评物品）
3. 有背景素材图（如科技感界面、室内环境）


### 第5问：选择背景色调

让用户从以下选项中选择：

1. 暖白/米色（百搭，适合大多数场景）
2. 深灰/黑色（适合效率、科技、工作流）
3. 柔和彩色（适合种草、生活类）
4. 模糊真实背景（适合教程、问答）
5. 纯色背景（极简风）


### 第6问：选择字体风格

让用户从以下选项中选择：

1. 粗黑体（冲击力强，最推荐）
2. 手写体（亲切感）
3. 科技感无衬线（适合AI/工具类）
4. 圆体（可爱、亲和）


### 第7问：选择字体颜色效果

让用户从以下选项中选择：

1. 柔和浅黄 #FDFFA7 + 黑色描边（品牌色，最推荐）
2. 纯白 + 黑色描边（干净百搭）
3. 黑色 + 白色描边（高级感）
4. 高亮荧光色（醒目突出）


### 第8问：从候选标题中选择最终标题

根据用户提供的素材，生成 **3个候选标题**，让用户选一个。

标题必须满足：
- 带数字（如：3个、5步、10分钟）
- 带结果（如：省下5000块、效率翻倍、告别拖延）
- 不超过15个字


## 输出格式

收集完所有信息后，按照用户选择的风格模板，输出以下格式的提示词：

```markdown
## 🎯 封面设计提示词

[风格名称] | [色调] | [情绪关键词]

**人物设定**：[描述人物姿势/表情/位置]

**标题文案**：[用户选定的标题]

**排版布局**：
- 主标题位置：[顶部/中部/底部]
- 人物位置：[居中/侧身/占比]
- 辅助元素：[贴纸/箭头/勾选清单]

**配色方案**：
- 主色：#XXXXX
- 强调色：#XXXXX
- 描边色：#000000

**AI绘图指令（英文）**：
[可直接复制到 Midjourney / Stable Diffusion 的完整提示词]

**视觉规范**：
- 比例：3:4
- 构图：高对比、高密度、一眼可读
- 安全区：文字和人物脸部不超出边界
